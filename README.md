# 🤖 AI 기반 PM 분석/설계 가이드 (AI-Analysis-Guide)

대규모 프로젝트 PM(Project Manager)이 AI 코드어시스턴트(Gemini, Copilot 등)를 활용하여 요구사항 분석부터 시스템 아키텍처, Front-End, Back-End, DB 설계에 이르기까지 체계적으로 설계 산출물을 작성할 수 있도록 돕는 실무 가이드 및 템플릿 프로젝트입니다.

## 🚀 사용 가이드 (How to Use)

1. 이 가이드는 **PM 및 아키텍트**가 AI 도구(Gemini, Copilot, ChatGPT 등)를 활용할 때 최적의 산출물을 출력하도록 프롬프트 가이드라인을 제공합니다.
2. 각 문서 내에 첨부된 **AI 프롬프트 템플릿**을 복사하여, 프로젝트 상황에 맞게 괄호 `[...]` 안의 세부 내용을 수정해 사용하십시오.
3. 로컬에서 가이드라인 웹 대시보드를 직접 렌더링하여 탐색하고 싶다면 [index.html](index.html) 파일을 웹 브라우저로 실행하시면 됩니다.
